# Calculator App

This project provides a convenient calculator application designed to help users perform calculations directly within their terminal environment, eliminating the need to switch between multiple windows.

## Features

- **Terminal-Friendly:** Perform calculations without leaving your command-line interface.
- **Easy to Use:** A straightforward interface ensures a smooth user experience.
- **Standard Functions:** Includes all essential arithmetic operations and standard calculator functionalities.
